# 영화 검색 사이트
<br>
          
## 🎄프로젝트 소개
Vue.js와 omdb 영화 API를 활용해 영화를 검색하고 영화의 상세 정보를 확인할 수 있는 사이트

<br>

## 📽프로젝트 사이트
- 배포 링크: https://goofy-kalam-f8b225.netlify.app
<br>
